# 通过源码安装

  • 下载源码
  • 编译
  • 设置系统变量
  • 测试

如果你已安装GO 1.18+版本 (opens new window)，你可以参照以下步骤编译源码生成Etcd-Carry。

# 下载源码

下载Etcd-Carry的源码压缩包 (opens new window)并进行解压缩，或者通过以下命令克隆Etcd-Carry的源码：

$ git clone https://github.com/etcd-carry/etcd-carry.git
1

# 编译

$ cd etcd-carry
$ make
1
2

生成的二进制文件在bin目录下。

# 设置系统变量

将bin目录的绝对路径设置到PATH系统变量，例如：

$ export PATH="$PATH:`pwd`/bin"
1

# 测试

测试下etcd-carry命令：

$ etcd-carry --help
1
